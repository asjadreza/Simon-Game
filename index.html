How To Play This Game

Simon is an electronic game of memory skill invented by Ralph H. Baer and Howard J. Morrison, working for toy design firm Marvin Glass and Associates, with software programming by Lenny Cope. The device creates a series of tones and lights and requires a user to repeat the sequence. Now this game is in front of you, you can start buy clicking the start button below with level 0, once you started the game it will produce different sound for different color and will blink at the same time. You have to see which button would blink, if you see it you have to click that colored button. If you do it correct you are increment to level 1. Again after this the button will blink and you have to see it and if see it you have to click in order of previous colored button you clicked and the colored button blinked in level 1. And have to continue like this and your level gets increases.
